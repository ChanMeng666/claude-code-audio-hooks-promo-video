# Promo Video Update for Claude Code Audio Hooks v4.7.0

## Context

The promotional video was built for v4.0.3 and shows 9 hook types, basic installation, and 2 audio sets. The project has since evolved to v4.7.0 with 22 hook types, Focus Flow, webhooks, async execution, smart matchers, snooze, and more. The video needs to reflect these changes while preserving the existing visual style and brand identity.

## Strategy

Iterative update on the existing 7-section structure: update data/copy in existing sections, add 2 new sections for major features. Target duration: 65-70 seconds (up from 57s).

## Updated Video Structure

```
Section:            Dur:    Transition:   Changes:
1. Intro            5.0s    → wipe        version v4.0.3 → v4.7.0
2. Problem          7.0s    → fade        unchanged
3. Solution         6.7s    → slide       badges updated
4. Features         16.0s   → fade        sub-scenes updated
5. AdvancedFeatures 10.0s   → slide       NEW (Focus Flow + Webhooks)
6. Demo             10.0s   → wipe        keep existing, user will replace later
7. Ecosystem        7.0s    → fade        NEW (Smart Matchers + Async + Snooze)
8. CrossPlatform    7.0s    → fade        counters updated
9. CTA              9.0s    (end)         copy + install command updated
```

Estimated total with transitions: ~68 seconds (~2040 frames at 30fps).
